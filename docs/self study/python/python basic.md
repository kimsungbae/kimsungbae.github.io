## 변수

### 변수
 - 데이터를 값을 저장하는 공간
 - 프로그램 작성시 데이터 값을 효율적으로 저장하고 관리하는 용도

#### 여러 변수를 동시에 한개 값에 할당

```
변수 a, b에 1의 값을 각각 저장
a = b = 1
```

#### 여러개의 변수에 각각의 값을 할당

```
a , b = 1, 2
```

#### 변수 관련 함수 or 연산자 등

| 함수 or 연산자 등 | 설명                                        | 출력값                         | 비고                                                               |
| :---------------- | :------------------------------------------ | :----------------------------- | :----------------------------------------------------------------- |
| id(변수명)        | 변수의 메모리 주소 출력                     | 주소값                         | -                                                                  |
| type(변수명)      | 변수의 타입을 출력                          | str, int, class, tuple, etc... | -                                                                  |
| 변수1 ia 변수2    | 변수1과 변수2가 같은 객체를 가리키는지 비교 | True or False                  | 숫자, 문자와 다르게 리스트는 생성될 때 마다 새로운 객체를 생성<br> |

```
ex) a = 2, b = 1 + 1
print(a is b) => True
c = [1, 2]
d = [1, 2]
print(c is d) => False
``` 

변수1 == 변수2: 변수1과 변수2가 같은 값인지 비교

```
a = 'koera'
b = a[:-1]
print(a)            => korea
print(id(a))        => (변수a의 주소값)
print(b)            => kore          
print(id(b))        => (변수b의 주소값)
print(a is b)       => False
```

#### 특수기호
  - 변수 선언할 때 변수 내에서 쌍따옴표("), 홑따옴표(') 사용 시, \(역슬래시)랑 같이 사용

#### 상수
 - 항상 변하지 않는 수(변수는 변하는 수)
 - 사용자가 임의로 바꾸거나 변경할 수 없음.(사전에 미리 정의되어 있음)
 - 숫자, 문자 등이 있음
 - 상수명은대문자로 표시

#### 튜플

```
1, 2를 한쌍으로 튜플 형식으로 변수 a, b에 각각 저장
a = b = 1, 2
print(a)                => (1, 2)
print(type(a))          => <class 'tuple'>
print(a[0], a[1])       => 1 2
print(type(a[0]))       => <class 'int>
```

#### 기타
 - 두 변수의 값 교환: `a, b = b, a`

### 리스트(list)

#### 리스트 슬라이싱
```
리스트[i_start:i_end:i_step]
* i_start : 생략시 0(인덱스의 시작)
* i_end   : 생략시 0(끝/-1)
* i_step  : 생략시 1(스텝)
```

#### 리스트에서 항목 삭제하기
```
del 리스트명[i]
* 리스트에서 인덱스가 i인 항목 삭제
```

#### 리스트에서 항목의 존재 여부 확인하기
```
항목 in 리스트
* True or False 반환
```

#### 리스트 메서드

| 메서드 | 설명 | 사용 예 |
|---|------|-------------------|
| append() | 리스트에서 항목 하나를 맨 마지막에 추가 | 리스트명.append('항목') |
| insert() | 리스트에서 특정 위치에 항목을 삽입 | 리스트명.insert(1, '항목') |
| extend() | 리스트에서 항목 여러 개를 맨 마지막에 추가 | 리스트명.extend(['항목1', '항목2']) |
| remove() | 입력값과 첫 번째로 일치하는 항목을 리스트에서 삭제 | 리스트명.remove('항목') |
| pop() | 리스트의 마지막 항목을 제거한 후에 반환 | 리스트명.pop() |
| index() | 리스트에서 인자와 일치하는 첫 번째 항목의 위치를 반환 | 리스트명.index('항목') |
| count() | 리스트에서 인자와 일치하는 항목의 개수를 반환 | 리스트명.count('항목') |
| sort() | 숫자나 문자열로 구성된 리스트 항목을 순방햐으로 정렬 | 리스트명.sort() |
| reverse() | 리스트 항목을 끝에서부터 역순으로 정렬 | 리스트명.reverse() |
| clear() | 리스트의 모든 항목을 삭제 (del 리스트명[:]) | 리스트명.clear() |

#### 리스트 컴프리헨션(list comprehension)
```
>>> squares = []
>>> for x in range(10):
···  squares.append(x**2)
···
>>> squares
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

```
squares = list(map(lambda x: x**2, range(10)))
```

```
squares = [x**2 for x in range(10)]
```

```
[(x, y) for x in [1, 2, 3] for y in [3, 1, 4] if x != y]
```

### 튜플(tuple)
리스트와 비슷하지만, 한번 입력하면 항목 변경 불가
항목을 변경하는 메서드가 아닌 다른 리스트 메서드는 튜플에서도 사용 가능

#### 튜플 생성 예
```
tuple1 = (1, 2, 3, 4)
tuple2 = 5, 6, 7, 8
tuple3 = (1,)
tuple4 = 2,
```

### 세트(set)
리스트와 튜플과 다른 점은 <b>데이터의 순서가 없고</b>, <b>데이터를 중복해서 쓸 수 없다</b>는 점

```
>>> set1 = {1, 2, 3, 3}
>>> print(set1)
··· {1, 2, 3}
```

#### 세트 메서드

| 메서드 | 기호 표시 | 사용 예 |
|--------------------------|-----------|----------------------|
| 교집합(intersection) | A ∩ B | A.intersection(B) or A & B |
| 합집합(union) | A ∪ B | A.union(B)  or A '| B |
| 차집합(difference) | A - B | A.difference(B) or A-B |

### 리스트, 튜플, 세트 변환
리스트, 튜플, 세트는 서로 변환이 가능
```
list(), tuple(), set()
```

### 딕셔너리(dictionary)
key:value 형태로 키와 값이 쌍을 이룬다<br>
key에는 숫자와 문자열, 값에는 숫자, 문자열, 리스트, 튜플, 딕셔너리 등 다양한 형태의 데이터 사용 가능

```
dict_name = {key1:value1, key2:value2, key3:value3 }
```

#### 딕셔너리 다루기
- 딕셔너리 추가 또는 기존값 수정
  - dict_name['key']='value'
- 딕셔너리 삭제
  - del dict_name['key']

| 메서드 | 설명 | 사용 예 |
|---|------|-------------------|
| key() | 딕셔너리에서 키 전체를 리스트 형태로 반환 | dict_name.keys() |
| values() | 딕셔너리에서 값 전체를 리스트 형태로 반환 | dict_name.values() |
| items() | 딕셔너리에서 키와 값의 쌍을 (키, 값)처럼 튜플 형태로 반환 | dict_name.items() |
| update(dict_name2) | 딕셔너리에서 딕셔너리 데이터(dict_name2) 추가 | dict_name.update(dict_name2) |
| clear() | 딕셔너리의 모든 항목 삭제 | dict_name.clear() |
